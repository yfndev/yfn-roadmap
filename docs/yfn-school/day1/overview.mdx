## 🎯 **Ziele des Tages**

1. Die Teilnehmer:innen lernen die Ziele und Struktur von YFN Schools kennen.  
2. Erste Inspiration durch Erfolgsgeschichten von Gründern.  
3. Einführung in das **Growth-Mindset** für Unternehmertum.  
4. Effektive **Teamarbeit** und erste Gruppenbildung.  
5. Praktische Einführung in das **Business Model Canvas**.
